Template matching Method Overview----模版匹配系列之综述

Author: Lance Date: 十月 5, 2017  07:17:14 Category: Computer Vision

Abstract: 模版匹配是一种很古老的方法，广泛应用于很多的场景。 传统的算法很简单也很容易实现，但缺点也是显而易见。由于近两年基于近邻的方法在模版匹配中取得非常不错的效果，个人觉得比基于data-driven的更接地气，也更稳定强大。 本篇文件将简要的概述模版匹配相关算法，包括传统的intuitive methods的以及最新的2015 CVPR 和 BBS和2017CVPR DDIS。 详细的算法解释和代码（不贴代码都是流氓）将会在后续的blog陆续放出，敬请期待～

1. SSD/SAD

Sum of Square Distance. \[ R(x,y)= \sum _{x',y'} (T(x',y')-I(x+x',y+y'))^2 \]

Sum of Absolute Distance \[ R(x,y)= \sum _{x',y'} |(T(x',y')-I(x+x',y+y')) |\]

2. NCC

Normalized Cross-Correlation \[ R(x,y)= \frac{\sum_{x',y'} (T(x',y') \cdot I(x+x',y+y'))}{\sqrt{\sum_{x',y'}T(x',y')^2 \cdot \sum_{x',y'} I(x+x',y+y')^2}} \]

3. EMD/HM

Earth Movers Distance(The EMD problem can be solved as a transportation problem)

Histogram Matching : Completely disregards geometry, which is a powerful cue.

4. DDIS(A improvement version on BBS)

  • DIS(Diversity similarity)

  • DDIS(Deformable Diversity Similarity)

Summary

  • It's all about how to measure the similarity in a feature space in order to handle the noises, illumination changes, geometric transformations, clutter background, non-rigidly deformations.

  • You can use the RGB information or deep features extracted from CNN to form the feature space.

  • The first three methods are very simple and easy to understand or implement.

  • DDIS and BBS use the NN match pairs as the similarity metric which can latch on to the relevant features that correspond between the template and target, instead of adding penalty on all pixel. DDIS use the one direction match methods rather while BBS use bi-directional match methods to get the performance boost.
